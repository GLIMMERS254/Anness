## 🌐 Deploy to GitHub Pages

### Step 1: Create GitHub Repository

1. Go to [github.com](https://github.com)
2. Click **"+"** → **"New repository"**
3. Name: `Glimmerweb` (or your choice)
4. Select **Public**
5. Click **Create repository**

### Step 2: Upload Files (Web Interface - Easiest)

1. Click **"Add file"** → **"Upload files"**
2. Select all 4 files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
   - `.gitignore`
3. Commit message: `Initial commit - Anne's website 💕`
4. Click **"Commit changes"**

### Step 3: Enable GitHub Pages

1. Go to **Settings** (top menu)
2. Click **"Pages"** (left sidebar)
3. Under **Build and deployment**:
   - Select branch: **main**
   - Select folder: **/ (root)**
4. Click **Save**
5. Wait 1-2 minutes

### Step 4: Your Website is Live!

Visit: `https://yourusername.github.io/Glimmerweb`

Example: `https://GLIMMERS254.github.io/Glimmerweb`

## 🎨 Customization

### Change Names
Edit `index.html`:
```html
<h1 class="hero-title">Ray & Anne</h1>
<p class="logo">Ray & Anne 💕</p>
```

### Change Colors
Edit `styles.css` - Find `:root`:
```css
:root {
    --primary-orange: #FF8C42;
    --dark-orange: #E65100;
    --light-orange: #FFB088;
}
```

Try these:
- **Red**: `#FF6B6B`, `#E63946`
- **Purple**: `#9D4EDD`, `#7209B7`
- **Pink**: `#FF006E`, `#FB5607`
- **Blue**: `#0077B6`, `#00B4D8`

### Change Timeline Stories
Edit `index.html` - Find `.timeline-item` sections:
```html
<h4>💑 How It Began</h4>
<p>Your story here...</p>
```

### Change Renewal Message
Edit `index.html` - Find `.renewal-message`:
```html
<p>I want you to know that I forgive you...</p>
```

## ❓ Troubleshooting

### Photos/Messages Not Saving
- Check if localStorage is enabled in browser
- Check browser's privacy settings
- Try different browser

### Website Not Live After Upload
- GitHub takes 1-2 minutes to build
- Wait and refresh
- Check that repo is PUBLIC

### Want to Fix Something?
1. Edit file locally
2. Upload to GitHub again
3. Commit changes
4. Website updates automatically (1-2 min)
